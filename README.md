# DarkBox WM

Minimalistic Wayland compositor for old netbooks.

## Features
- Pure black background
- Only terminals
- System monitor in top-right corner
- Keyboard-driven workflow

## Keybindings
- `Super + Enter` - New terminal
- `Super + j/k` - Switch focus
- `Super + h/l` - Resize windows
- `Super + Shift + c` - Close window
- `Super + q` - Quit

## Installation
```bash
git clone https://github.com/wiznkingpro/darkbox-wm
cd darkbox-wm
make
sudo make install
